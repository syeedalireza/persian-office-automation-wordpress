A comprehensive office automation and correspondence management system (Dabirkhane) for WordPress.

== Description ==

Persian Office Automation SaaS is a powerful plugin designed to transform your WordPress website into a full-featured office automation system. It manages incoming and outgoing letters, internal correspondence, and tasks with a modern and user-friendly interface.

Key Features:
*   **Correspondence Management:** Manage incoming and outgoing letters with ease.
*   **Internal Notes:** Send and receive internal memos.
*   **Task Management:** Assign tasks to users and track their progress.
*   **Jalali Date Support:** Full support for Persian calendar (Jalali).
*   **Modern Dashboard:** A beautiful and responsive dashboard for quick insights.
*   **Role Management:** Custom roles and capabilities for granular access control.
*   **Digital Archive:** Archive letters and documents securely.

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/office-automation` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Navigate to the "Persian Office Automation" menu to start configuring the system.
4.  Ensure your user role has the necessary capabilities (Admins have full access by default).
